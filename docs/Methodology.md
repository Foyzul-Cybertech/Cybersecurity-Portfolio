# Methodology — Penetration Testing Lifecycle

This document outlines the structured approach used across all cybersecurity projects.

---

## 1. Reconnaissance
Collecting information about the target system using passive and active techniques.

Examples:
- IP discovery
- Domain enumeration
- Service identification

---

## 2. Enumeration
Identifying specific details about services, users, and system configuration.

Examples:
- Port scanning
- Service version detection
- Directory enumeration

---

## 3. Exploitation
Attempting to exploit identified vulnerabilities.

Examples:
- SQL Injection
- XSS attacks
- Service exploitation using Metasploit

---

## 4. Post-Exploitation
Maintaining access and analyzing compromised systems.

Examples:
- Privilege escalation
- Credential harvesting
- System exploration

---

## 5. Reporting
Documenting findings, impact, and remediation steps.

Includes:
- Vulnerability description
- Evidence (screenshots/logs)
- Risk level
- Fix recommendations

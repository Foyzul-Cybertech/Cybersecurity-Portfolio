# Network Analysis (Wireshark Security Labs)

## 📌 Overview
This section focuses on network traffic analysis using Wireshark to identify normal behavior, detect vulnerabilities, and simulate attack patterns.

## 🎯 Objective
Develop skills in packet inspection, traffic analysis, and detection of malicious network behavior.

## 🧪 Labs Included
- Baseline Network Traffic Profiling & Protocol Analysis
- Credential Exposure in HTTP Traffic
- Suspicious Network Activity & Attack Detection

## 🛠️ Tools Used
- Wireshark
- Kali Linux
- DVWA (for traffic generation)
- Nmap (for simulation)

## 🧠 Key Skills Demonstrated
- Packet-level traffic analysis
- Protocol identification (TCP, DNS, HTTP, TLS)
- Detection of insecure transmissions
- SOC-style traffic investigation

## 📊 Methodology Summary
All labs follow a structured approach:
Reconnaissance → Traffic Capture → Packet Analysis → Findings → Mitigation

- Network Traffic Analysis (Wireshark) — packet inspection, protocol analysis, traffic investigation  
- Penetration Testing (Metasploit, Burp Suite) — exploitation, vulnerability testing, web app attacks  
- Web Application Security (DVWA, Juice Shop) — OWASP Top 10 vulnerabilities (SQLi, XSS, authentication flaws)  
- Scripting & Automation (Python) — security tools, scanning scripts, data analysis  
- Capture The Flag (CTF) Problem Solving — structured attack methodology and exploitation thinking  
- Security Lab Development — virtual environments for controlled testing and simulations

Within each topic, projects are organized by real-world tasks and scenarios, such as:

Each project includes:

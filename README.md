- Network Traffic Analysis (Wireshark) — packet inspection, protocol analysis, traffic investigation

- Penetration Testing (Metasploit, Burp Suite) — exploitation, vulnerability analysis, web testing

---

## Methodology

All projects follow a structured penetration testing lifecycle:

1. Reconnaissance — information gathering  
2. Enumeration — identifying services and attack surfaces  
3. Exploitation — leveraging vulnerabilities  
4. Post-Exploitation — privilege escalation and system analysis  
5. Reporting — documenting findings and remediation steps  

Full breakdown available in `/docs/methodology.md`

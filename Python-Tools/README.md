# Python Tools

## 📌 Overview
This section contains Python-based tools developed to automate cybersecurity tasks such as network scanning, packet analysis, and reconnaissance workflows.

To demonstrate programming skills in cybersecurity by building automation tools that support penetration testing and network analysis tasks.

- Service detection via banner grabbing
- Fast network reconnaissance

---

### 🌐 Recon Automation Toolkit
- IP range scanning
- DNS resolution and lookup automation
- Open port discovery across targets
- Combined reconnaissance workflow

### 🧠 Security Framework (Mini Tool Suite)
A modular Python-based security toolkit:

- scanner.py → network scanning engine  
- sniffer.py → packet capture and analysis  
- report_generator.py → structured output reporting

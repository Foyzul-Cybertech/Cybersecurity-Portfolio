# Web Application Security Labs (OWASP / Burp Suite)

## 📌 Overview
This section focuses on identifying and exploiting common web application vulnerabilities using OWASP methodology.

## 🎯 Objective
Understand how web applications are attacked and how vulnerabilities such as SQL Injection and XSS are exploited.

## 🧪 Labs Included
- OWASP Core Vulnerabilities Lab
- Burp Suite Interception & Manipulation Lab
- Full Web Application Pentest Report

## 🛠️ Tools Used
- Burp Suite
- DVWA
- OWASP Juice Shop

## 🧠 Key Skills Demonstrated
- Web request interception
- Parameter manipulation
- Authentication testing
- OWASP Top 10 exploitation

## 📊 Methodology Summary
Recon → Endpoint discovery → Vulnerability testing → Exploitation → Reporting

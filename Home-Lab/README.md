# Home Lab Security Environment

---

## 📌 Overview
This section simulates real-world cybersecurity environments using virtual machines to perform both offensive and defensive security operations.

---

## 🎯 Objective
To build practical experience in attack simulation, network monitoring, and incident detection in a controlled environment.

---

## 🧪 Labs Included

### 🧱 Basic Home Lab Setup
- VirtualBox / VMware configuration
- Kali Linux attacker machine
- Windows target machine
- Metasploitable test environment
- Isolated virtual network setup

---

### ⚔️ Attack vs Defense Simulation Lab
- Simulated attacks from Kali Linux
- Traffic monitoring using Wireshark
- Windows Event Log analysis
- Detection of malicious activity patterns

---

### 🛡️ SOC Mini Incident Detection Lab
- Simulated brute force attacks
- Port scanning detection
- Log analysis and correlation
- Basic incident response workflow
- Structured reporting of findings

---

## 🛠️ Tools Used
- VirtualBox / VMware  
- Kali Linux  
- Windows VM  
- Wireshark  
- Event Viewer  

---

## 🧠 Key Skills Demonstrated
- Virtual lab setup and configuration  
- Attack simulation and monitoring  
- Log analysis and incident detection  
- SOC-style investigation workflow  

---

## 📊 Methodology Summary
Environment Setup → Attack Simulation → Monitoring → Detection → Analysis → Reporting
